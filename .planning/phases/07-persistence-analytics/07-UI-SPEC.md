## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#0D0D0D` | Page background, main surface |
| Secondary (30%) | `#1A1A1A` | Cards, panels, counter backgrounds |
| Accent (10%) | `#00FF88` | Counter numbers, LIVE badge, active states, positive indicators |
| Secondary Accent | `#FFB800` | Medium-severity warnings, secondary highlights |
| Destructive | `#FF3B5C` | Error states, critical alerts only |

Accent reserved for:
- LED counter numbers (all counter values)
- LIVE badge glow animation
- Positive status indicators
- Primary interactive hover states

Secondary Accent (`#FFB800`) reserved for:
- Medium-severity attack labels (e.g., warning-level threat indicators)
- Secondary highlights where phosphor green would be too prominent

Destructive (`#FF3B5C`) reserved for:
- Critical alert states
- Error messages only

## Phase 7 Specific Design Specifications

### New StatsPanel Sections

The existing StatsPanel displays archetype counters in a horizontal flex row. This phase extends it with three new sections below the archetype row:

#### Section 1: Lifetime Attacks Counter

| Property | Value |
|----------|-------|
| Label | "Lifetime" |
| Value | Cumulative attack count since deployment |
| Format | Locale-formatted number (e.g., `1,247`) |
| Cap | Display as `99,999+` above threshold |
| Component | Reuse existing `CounterBox` |
| Position | First in analytics row (leftmost) |

#### Section 2: Top Countries List

| Property | Value |
|----------|-------|
| Label | "Top Countries" |
| Display | Top 5 countries by attack count |
| Format | Country flag emoji + country name + count |
| Example | `🇨🇳 China: 247` |
| Sort | Descending by count |
| Component | New `CountryList` component (styled consistently with CounterBox aesthetic) |
| Position | Second in analytics row |
| Max items | 5 |

#### Section 3: Attack Methods Panel

| Property | Value |
|----------|-------|
| Label | "Attack Methods" |
| Sub-sections | Protocol breakdown (SSH/Telnet) + Top Ports |
| Protocol format | `SSH: {count}` / `Telnet: {count}` |
| Ports format | Port number + count (e.g., `22: 342`) |
| Max ports | 5 |
| Component | New `MethodsPanel` component |
| Position | Third in analytics row (rightmost) |

### Layout Structure

```
StatsPanel
├── Row 1: Archetype Counters (existing)
│   ├── Total
│   ├── Script Kiddies
│   ├── APT Operatives
│   ├── Botnet Drones
│   ├── IoT Worms
│   └── Hacktivists
│
└── Row 2: Analytics Row (new)
    ├── Lifetime Counter (CounterBox)
    ├── Top Countries (CountryList)
    └── Attack Methods (MethodsPanel)
        ├── SSH/Telnet counts
        └── Top 5 Ports
```

### Visual Consistency

| Element | Style |
|---------|-------|
| Counter backgrounds | `#0a0a0a` (dark LED panel) |
| Counter borders | `var(--color-border)` |
| Counter text | Phosphor green (`#00FF88`) with glow text-shadow |
| Counter labels | Muted text (`#6B6B6B`), 11px mono |
| Section spacing | `gap-md` (16px) between rows |
| Row flex direction | `flex-row` with `flex-wrap` |

### Animation

| Element | Animation |
|---------|-----------|
| Counter value change | Scale pulse (existing CounterBox behavior): `[1, 1.02, 1]` over 150ms |
| New sections entrance | Fade-in with stagger (50ms delay between items) |
| Country/method count updates | Same scale pulse as archetype counters |

### Data Flow

| Event | Frontend Action |
|-------|-----------------|
| `attack_history` (on connect) | Replace entire attack list + update all analytics |
| `attack_event` (new attack) | Append to list, increment counters, update analytics aggregations |
| Connection error | Display "SIGNAL LOST" banner (existing behavior) |
