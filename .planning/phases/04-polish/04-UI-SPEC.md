## Color

### Dark Mode (Primary)

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#0D0D0D` | Page background |
| Secondary (30%) | `#1A1A1A` | Surface (cards, panels) |
| Secondary Raised | `#222222` | Elevated surfaces (sidebar) |
| Accent (10%) | `#00FF88` | LIVE badge, active states, positive events, primary CTAs |
| Accent Dim | `#00CC6A` | Hover state for primary accent |
| Accent Glow | `rgba(0, 255, 136, 0.15)` | Glow/shadow effects |

### Light Mode (Toggle)

| Role | Value | Usage |
|------|-------|-------|
| Background | `#F5F4F0` | Page background |
| Surface | `#FFFFFF` | Cards, panels |
| Surface Raised | `#FAFAFA` | Elevated surfaces |
| Border | `#E0DDD5` | Default borders |
| Text Primary | `#1A1A1A` | Primary text |
| Text Muted | `#6B6B6B` | Secondary text |
| Accent (desaturated 15%) | `#00DB75` | Primary accent in light mode |

**Accent reserved for:** LIVE badge, connection status indicators, counter values (phosphor green glow), positive state indicators, primary CTAs.

**Secondary accent (Amber):** `#FFB800` — Warning-level threats, secondary highlights, hover states on non-primary elements.

**Alert (Danger):** `#FF3B5C` — Critical attacks, error states, SIGNAL LOST banner.

**Source:** DESIGN.md Color section, CONTEXT.md D-11

## Light Mode Toggle

Per CONTEXT.md D-09 through D-12:

- **Placement:** Header, next to ConnectionStatus component
- **Icon:** Sun/Moon SVG inline (no icon library)
- **Behavior:** CSS custom properties switch via class on `<html>` or `<body>`
- **Default:** Dark mode on each page load (no localStorage persistence)
- **No `prefers-color-scheme` detection** — explicit toggle only

**Implementation:**
1. Add `.light` class to `<html>` element when toggle is active
2. Define light mode tokens in globals.css under `[data-theme="light"]` or `.light` selector
3. Toggle button updates DOM class and internal state

---

## Mobile Stats Bottom Sheet

Per CONTEXT.md D-05 through D-08:

- **Trigger:** "Stats" button in header (visible only on mobile breakpoint <768px)
- **Content:** StatsPanel content (CounterBox row with flex-wrap)
- **Animation:** Slide-up from bottom with backdrop
- **Dismiss:** Tap outside backdrop OR swipe down gesture
- **Backdrop:** Semi-transparent overlay (`rgba(0, 0, 0, 0.5)`)

**Framer Motion implementation:**
- `AnimatePresence` with `initial={{ y: "100%" }}`, `animate={{ y: 0 }}`, `exit={{ y: "100%" }}`
- Backdrop fade: `initial={{ opacity: 0 }}`, `animate={{ opacity: 1 }}`, `exit={{ opacity: 0 }}`
- Duration: 300ms
