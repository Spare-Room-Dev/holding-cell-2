## Phase 2 Component Specifications

### JailCellGrid

| Property | Value |
|----------|-------|
| Layout | 70% width, full viewport height minus header |
| Background | CSS-only stone/brick texture using gradients |
| Overlay | Iron bars via CSS `repeating-linear-gradient` |
| Capacity | 20 prisoners (fixed grid slots) |
| Prisoner order | Bottom-up stack, newest at top (LIFO) |
| Prisoner size | 56px square placeholder boxes |
| Grid gap | 8px (`--spacing-sm`) |
| Fade-out timing | 250ms `ease-out` (short duration from DESIGN.md) |
| Empty state | Centered text with CRT scanline overlay |

**Prisoner Placeholder (Phase 2):**
- 56px square div
- Background: archetype color with 20% opacity fill
- Border: 2px solid archetype color
- Rounded corners: `--radius-md` (8px)
- No hover interaction (Phase 3 feature)

### StatsPanel

| Property | Value |
|----------|-------|
| Layout | 30% width sidebar, scrollable |
| Counter layout | Horizontal row, 6 counters side-by-side |
| Counter style | Digital LED display boxes |
| Counter update | Phosphor green glow on increment |
| Number format | Locale commas, cap display at "99,999+" |
| Overflow scroll | Independent scroll if content exceeds viewport |

**Counter Box Design:**
- Dark background (`#0D0D0D` or `#1A1A1A`)
- Monospace font (IBM Plex Mono, `font-mono`)
- Number display: 14px (`--text-mono`)
- Label: 11px (`--text-mono-sm`)
- Glow: `phosphor-glow` utility on value change

### Layout Split

| Region | Width | CSS Class |
|--------|-------|-----------|
| JailCellGrid | ~70% | `flex-[7]` or grid `col-span-7` |
| StatsPanel | ~30% | `flex-[3]` or grid `col-span-3` |

**Source:** CONTEXT.md D-01 to D-20

## Integration Points

| Component | Data Source | Pattern |
|-----------|-------------|---------|
| JailCellGrid | `useSocket().state.attacks` | Read-only, derive visible prisoners |
| StatsPanel | `useSocket().state.attacks` | Read-only, count by archetype |
| Both | `useSocket()` from SocketContext | Established useReducer pattern |

**Derived State:**
- Prisoners to display: `attacks.slice(0, 20)`
- Counts: `{ total, byArchetype }` computed from attacks array
